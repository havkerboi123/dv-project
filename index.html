Project Description

This project presents an interactive data visualization dashboard for analyzing global YouTube statistics, providing insights into the world's top content creators, their geographic distribution, earnings potential, and content categories. The dashboard enables users to explore YouTube's ecosystem through interactive visualizations that respond dynamically to user selections.

The project consists of two main components:

  • Interactive Dashboard: A Streamlit-based web application with real-time filtering and cross-visualization interactivity. Users can filter data by clicking on map regions or selecting points in scatter plots, with all visualizations updating simultaneously.
  • Exploratory Data Analysis (EDA): Statistical analysis and visualization scripts for understanding data distributions and relationships.

Dataset Description

Dataset Name: Global YouTube Statistics.csv

Total Records: 996 YouTube channels

Encoding: Latin-1

Key Attributes

The dataset contains comprehensive information about the world's top YouTube channels, including:

Channel Information
  • rank, Youtuber name
  • Title, category
  • channel_type
Performance Metrics
  • subscribers, video views
  • uploads, rankings
  • Recent engagement
Financial Data
  • Earnings estimates
  • Monthly & yearly ranges
Geographic Data
  • Country, coordinates
  • Demographics

Data Quality

  • Missing values handled in Country and category fields (filled with 'Unknown'/'Misc')
  • Rows with missing creation year removed for temporal analysis
  • Derived metric: Avg_Yearly_Earnings calculated as average of lowest and highest yearly earnings

Dataset Overview

The dataset represents a comprehensive snapshot of the global YouTube creator ecosystem, focusing on top-performing channels across various metrics. Channels span multiple countries worldwide, with geographic coordinates enabling mapping visualizations. Content categories include Music, Entertainment, Gaming, Education, People & Blogs, Shows, Sports, Film & Animation, and more.
